Primeiro trabalho de programação que foi pedido na cadeira de programação.

    Objetivo - Realizar um jogo de batalha naval com regras de bastante especificas

    Opinião - Projeto bastante interresante, deu para conhecer o como funcionava o básico da linguagem de programação C, não passado dos básicos

    Conceito Utilizados - Foram utilizados neste trabalho diversos conceitos que são aprendidos no inicio de uma carreira de programação, nomeadamente, como fazer funções, ter um código em C divido em vários ficheiros, tirar partido do uso da Makefile. Toda a memória que foi utilizada era memória estática, nunca tendo sido utilizado qualquer tipo de Alloc.
